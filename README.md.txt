#Pré-requisitos
Para que seja possivel rodar essa aplicação é necessário:
-Java 1.8.0
-Maven 3.8.3
-Chrome Versão acima da 99

#Inicializando
- Ao abrir o projeto de um refesh no maven ou instale as dependências com o 
seguinte comando no terminal "mvn clean install";
- Abra a pasta src;
- Abra a pasta java;
- Abra a pasta tests;
